# Bicycle Shop
The Bicycle Shop is an SPA project consisting of a client-side and an admin panel that remains hidden until the user logs in as an administrator or owner. Regular users do not have access to the admin panel. Similarly, administrators do not have access to the role allocation page on this site and can only edit cards and create new ones.
To order selected items, visitors must register, and they can also leave reviews on the site.
